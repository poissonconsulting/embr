# Removes comments from the template strings(s) of an mb object or a character vector.

Removes comments from the template strings(s) of an mb object or a
character vector.

## Usage

``` r
rm_comments(object, ...)
```

## Arguments

- object:

  The mb object or character vector.

- ...:

  Unused.

## Value

The mb object or character vector without comments in its template
string(s).
